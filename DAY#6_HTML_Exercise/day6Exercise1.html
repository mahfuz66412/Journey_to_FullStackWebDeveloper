What I’ve Learned in My HTML Journey

  1. Frontend (Client-side)

    • HTML – Structure of web pages
    • CSS – Styling and layout
    • JavaScript – Interactivity
    • TypeScript – A typed version of JavaScript (optional but helpful)
    • SASS / SCSS – CSS preprocessor (optional)
    • JSX – JavaScript + HTML (used in React)

  2. Backend (Server-side)

    • Node.js – JavaScript runtime for backend
    • Express.js – Server-side framework (built on Node.js)

  3. Database Languages

    • MongoDB Query Language (MQL) – NoSQL document database
    • SQL – Structured Query Language (optional for relational DBs)

  4. And others also
